# ALK Wallet Carmen

Bienvenido a ALK Wallet Carmen, una aplicación web simple para gestionar tu cartera digital.

## ¿Qué puedes hacer aquí?

- **Iniciar sesión** con tu cuenta
- **Depositar dinero** en tu cartera
- **Enviar dinero** a otros usuarios
- **Ver tus transacciones** y historial de movimientos
- **Acceder al menú principal** para navegar entre las opciones

## Estructura del proyecto

```
proyecto alk wallet carmen/
├── html/                 # Páginas HTML
│   ├── login.html       # Página de inicio de sesión
│   ├── menu.html        # Menú principal
│   ├── index.html       # Página de inicio
│   ├── deposit.html     # Depositar dinero
│   ├── sendmoney.html   # Enviar dinero
│   └── transactions.html # Ver transacciones
└── assets/              # Recursos
    ├── css/
    │   └── styles.css   # Estilos de la aplicación
    └── js/
        └── script.js    # Lógica de la aplicación
```

## Cómo usar

1. Abre el archivo `index.html` en tu navegador
2. Inicia sesión o navega a través del menú
3. Explora las diferentes funciones disponibles

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
